LogWrapper - version 1.0
LogWrapper utilise la réflection pour déterminer si les logs doivent être
envoyées à Log4j 2 ou dans la console.

LogWrapper-src.7z - archive contenant toutes les sources pour compiler la
bibliothèque. Contient :
  LogWrapper.java
  LogConsoleWrapper.java
  Log4j2Wrapper.java
  COPYING.txt
  COPYING-LESSER.txt
  lisezmoi.txt
  readme.txt

LogWrapper.java - classe générique mère de tous les wrappers.

Log4j2Wrapper.java - wrapper pour l'utilisation de Apache Log4j 2.

LogConsoleWrapper.java - wrapper pour l'affichage des logs dans la console.

COPYING.txt - la licence GPL version 3.

COPYING-LESSER.txt - la licence Lesser GPL version 3. Elle s'applique à
l'ensemble de ces fichiers.

lisezmoi.txt - description de LogWrapper en français.

readme.txt - description de LogWrapper en anglais.
